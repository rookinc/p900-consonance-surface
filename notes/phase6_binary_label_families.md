# Phase 6 Binary Label Family Audit

Status: sandbox hypothesis

Phase 6 compares several binary 0/30 edge-label families.

Because 0 and 30 are closed under cycle sums modulo 60, binary labels eliminate orientation drift by construction.

The useful discriminator is therefore not drift. It is the balance between identity closure and half-turn closure across G15 cycles.

This phase searches for a more consonant binary surface labeling before M-guided or Q-guided interfaces are introduced.
