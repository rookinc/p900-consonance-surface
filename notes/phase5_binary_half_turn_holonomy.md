# Phase 5 Binary Half-Turn Holonomy

Status: sandbox hypothesis

Phase 5 tests a G30-style binary orientation grammar.

Each G15 edge carries either:

    0 mod 60  = identity
    30 mod 60 = half-turn

This asks whether the P900 surface can eliminate orientation drift by restricting interface shifts to the sign/half-turn register.

This is a baseline, not a proven P900 law.
